Quick Installation

# Clone the repository
git clone https://github.com/DEVOLUTION-BOT/DEVOLUTION-MD1
# Install dependencies
cd DEVOLUTION-MD1 && npm install
# Configure the bot
cp config.example.js config.js && nano config.js
# Run the bot
npm start
